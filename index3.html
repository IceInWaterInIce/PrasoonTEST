Key concept in machine learning

  • Data : The fuel for ML. Data can be structured (tables, numbers) or unstructured (images, text, audio).

  • Features : The input variables used to make predictions.

  • Model : The mathematical representation of the relationship between inputs and outputs.

  • Training : The process of teaching a model using known data.

  • Testing : Checking the modelâ€™s accuracy on unseen data.

  • Prediction : The output the trained model generates when given new data.
